# Advent of Code 2025 in Charly
>  [Associated Blog Article](https://leonard.swiss/blog/aoc-2025/)

This is my attempt at solving each puzzle of the [Advent of Code 2025](https://adventofcode.com/)
in my own programming language [Charly](https://github.com/KCreate/charly-vm/).
Because the current state of the implementation lacks many
important features (e.g. HashMaps, a reasonable stdlib), implementing those missing parts
becomes part of the challenge.

The experience of solving each puzzle is documented in a dedicated blog article.
